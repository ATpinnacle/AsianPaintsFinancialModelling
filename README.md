Asian Paints Financial Modelling

A three-statement financial model and DCF valuation of Asian Paints Ltd.


⚠️ The Excel model itself is kept private and is not included in this repo. This README and the PDF attached above documents what the model contains.

What's Inside


Historical Financials - Profit & Loss, Balance Sheet, Cash Flow (annual + quarterly), cleaned and restated from raw filings
Ratio Analysis - growth, margin, and profitability ratios across historical years
Common Size Statement - statements as a % of revenue
Sales & EBIT Forecasting - forward projections built off historical trends
Intrinsic Growth / ROIC - reinvestment and return-on-capital calculations
WACC - cost of capital, built using a peer set (Berger Paints, Kansai Nerolac, JSW Dulux, Indigo Paints, Sirca Paints)
DCF Valuation - free cash flow projection, discounting, and intrinsic value/share price output
Peer Comps - relative valuation cross-check against listed peers

Tools and Sources Used

Microsoft Excel, public financial data (Screener, Top Stock Research, Yahoo Finance, etc.)
